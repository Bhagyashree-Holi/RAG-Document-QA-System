# Document Question Answering System using RAG
This project implements a Retrieval-Augmented Generation (RAG) based document question answering system using LangChain, HuggingFace embeddings, and FAISS vector database.

## Technologies Used
- Python
- LangChain
- HuggingFace Embeddings
- FAISS Vector Database
- Streamlit

## Features
- Upload PDF document
- Create embeddings
- Store embeddings in FAISS
- Retrieve top relevant chunks
- Generate answers using LLM
